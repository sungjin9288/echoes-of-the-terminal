*작성: AI 수석 내러티브 디자이너 / 대상 독자: 구현 담당 프로그래머*

## 1. Game Overview & Mechanics

### 세계관 요약 (3줄)

> 2142년, 인류 최후의 초거대 AI '아르고스(Argos)'가 자아를 획득하고 모든 글로벌 인프라를 장악했다.
> 저항군의 마지막 다이버들은 낡은 터미널을 통해 아르고스의 내부 방어 체계—의미론적 방벽(Semantic ICE)—에 침투를 시도한다.
> 아르고스는 역사 기록물을 위조한 함정 데이터를 방벽으로 사용하며, 다이버는 텍스트 속 논리적 모순을 찾아 키워드를 입력하는 것으로 방벽을 붕괴시킨다.

---

### 변주 시스템 (Mutators)

특정 스테이지 또는 플레이어 선택에 의해 활성화되며, 기본 루프에 전략적 난이도를 추가한다.

| ID | 이름 | 효과 |
|----|------|------|
| M-01 | **타임 프레셔 (Time Pressure)** | 각 텍스트 로그에 카운트다운 타이머(기본 90초)가 활성화된다. 시간 초과 시 답을 입력하지 못해도 Trace Level이 15 자동 상승한다. |
| M-02 | **리댁션 (Redaction)** | 텍스트 로그 내 단어의 약 20%가 `█████` 블록으로 마스킹되어 표시된다. 플레이어는 남은 문맥만으로 모순을 추론해야 한다. |
| M-03 | **노이즈 인젝션 (Noise Injection)** | 시대 오류 또는 논리 모순과 무관하지만 의심스럽게 보이는 가짜 키워드 2개가 텍스트에 추가 삽입된다. 오답 입력 시 페널티율이 1.5배 적용된다. |

---

### 성장 시스템 (Meta-Progression)

런 종료 후 획득하는 '시냅스 포인트(SP)'로 영구 특성을 해금한다.

| ID | 특성명 | 효과 |
|----|--------|------|
| P-01 | **오류 허용 버퍼 (Error Buffer)** | 오답 입력 시 Trace Level 상승량을 최대 50% 감소시킨다. (단계적 해금: 10% → 25% → 50%) |
| P-02 | **독해 보조 (Lexical Assist)** | 텍스트 로그를 일정 시간 이상 열람하면, 모순이 포함된 문장이 미묘하게 다른 색상으로 하이라이트된다. |
| P-03 | **역추적 프로토콜 (Backtrack Protocol)** | 스테이지 실패(사망) 시, 동일 스테이지를 Trace Level 50% 상태로 런당 최대 1회 재시작할 수 있는 기회를 부여한다. |

## 2. JSON Data Schema

각 추리 시나리오를 구성하는 단일 노드(Node)의 데이터 구조 정의.

```json
{
  "node_id": 0,
  "theme": "A",
  "difficulty": "Easy",
  "text_log": "플레이어에게 표시되는 텍스트 로그 본문. 개행은 \\n 사용.",
  "target_keyword": "플레이어가 터미널에 정확히 입력해야 하는 단어",
  "logical_flaw_explanation": "해당 키워드가 모순인 이유. 내부 설계 메모용으로 게임에 노출되지 않음.",
  "penalty_rate": 20
}
```

**필드 정의**

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `node_id` | `int` | 노드 고유 식별자 (1부터 순차 부여) |
| `theme` | `str` | 테마 코드. `"A"` \| `"B"` \| `"C"` \| `"D"` |
| `difficulty` | `str` | 난이도. `"Easy"` \| `"Hard"` |
| `text_log` | `str` | 플레이어에게 표시되는 텍스트 로그 전문 |
| `target_keyword` | `str` | 정답 키워드. 단일 단어여야 하며 `text_log` 내에 반드시 존재해야 함 |
| `logical_flaw_explanation` | `str` | 모순 근거 설명. 프로그래머/디자이너용 내부 메모 |
| `penalty_rate` | `int` | 오답 1회당 Trace Level 상승량 (단위: %). Easy=20, Hard=40 |
