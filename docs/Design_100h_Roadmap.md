# Design_100h_Roadmap.md
## Echoes of the Terminal — 100시간 플레이타임 설계 로드맵
*목표: 시나리오 수가 아닌 '시스템 승수'로 100시간을 확보한다.*

---

## 핵심 공식

```
총 플레이타임 = 노드 기반 콘텐츠(시간) × 시스템 승수
```

| 단계 | 노드 수 | 기반 시간 | 시스템 승수 | 예상 총합 |
|------|---------|-----------|------------|---------|
| MVP  | 18      | ~8h       | ×1.0       | ~8h     |
| v1.0 | 80      | ~35h      | ×1.5       | ~50h    |
| **v1.5 (목표)** | **120** | **~50h** | **×2.0** | **~100h** |

> **승수의 의미:** 같은 노드를 다른 조건·캐릭터·난이도로 다시 플레이하도록 만드는 시스템들의 합산 배율.

---

## LAYER 1 — 콘텐츠 볼륨 (기반 시간 확보)

### 목표 노드 수: 120개

현재 18개에서 아래 비율로 확장.

```
Easy      : 40개  × 평균 3분  = 120분
Hard      : 50개  × 평균 8분  = 400분
NIGHTMARE : 20개  × 평균 18분 = 360분
Special   : 10개  (아래 별도 정의)
──────────────────────────────────────
단순 합산:  880분 = 약 14.7시간 (1회 전체 노출 기준)
런 1회 평균 45분 × 재시도 포함 × 3 = 약 50시간 기반
```

### 신규 테마 생산 계획 (Data Factory 연장)

| 팩 | 테마 | 노드 |
|----|------|------|
| Pack 03 | 남극 연구 기지 / 르네상스 궁정 / 근미래 법정 / AI 내부 로그 | +10 |
| Pack 04 | 전쟁 포로 심문 기록 / 고대 이집트 사제 일지 / 사이버 테러 수사 / 식민지 행성 보고서 | +10 |
| Pack 05 | 정신병원 환자 일지 / 조선시대 사초 / 마피아 내부 문서 / 잠수함 항해 일지 | +10 |
| Pack 06~10 | (위 팩 패턴 반복, 테마 교차) | +50 |
| **Special 노드** | 아래 LAYER 2 참조 | +10 |

## LAYER 2 — 시스템 승수 (★ 재방문 동기 설계)

### 2-1. 다이버 클래스 시스템 (×1.5 승수)

플레이어블 캐릭터를 3종으로 분류. 각 클래스는 **같은 노드를 다른 방식으로 경험**하게 만든다.

| 클래스 | 코드명 | 특성 | 플레이 변화 |
|--------|--------|------|-----------|
| 분석가 | ANALYST | 오답 페널티 -30%, 타이머 없음 | 느리고 안전하게. 입문자·스토리 집중형 |
| 고스트 | GHOST | 타이머 -50% (압박), 정답 시 Trace -15 | 빠르고 위험하게. 스피드런 지향형 |
| 크래커 | CRACKER | 노드 진입 시 target_keyword 1글자 힌트 공개, 단 오답 페널티 +50% | 도박형. 고위험-고보상 |

> **승수 근거:** 같은 80개 노드를 3개 클래스로 플레이하면 순수 콘텐츠가 3배처럼 느껴진다. 실제 클리어 소요 시간도 클래스별로 다름.

---

### 2-2. 각성 레벨 시스템—ASCENSION (×1.8 승수)

Slay the Spire의 Ascension과 동일한 구조. 클리어할수록 영구적 난이도 가중치가 누적된다.

```
ASCENSION 0   : 기본 게임
ASCENSION 1   : 전체 penalty_rate +5
ASCENSION 2   : Easy 노드에서 Mutator M-03(노이즈 인젝션) 강제 활성
ASCENSION 3   : 타이머 기본값 -20초
ASCENSION 4   : NIGHTMARE 보스 도중 화면이 1회 5초간 블랙아웃
ASCENSION 5   : 런 시작 시 Trace Level이 20에서 출발
...
ASCENSION 20  : 모든 조건 누적. 아르고스 전용 "오염된 로그" 노드 추가
```

> 각성 레벨 20단계 × 3개 클래스 = **60개 클리어 목표** → 최소 60런 이상 동기 부여.

---

### 2-3. 유물 시스템—ARTIFACTS (×1.4 승수)

런마다 랜덤하게 획득하는 패시브 아이템. 빌드 다양성을 폭발적으로 늘린다.

**유물 예시 (15종 이상 목표):**

| 유물명 | 효과 | 트레이드오프 |
|--------|------|------------|
| 마모된 사전 | text_log 내 시대 오류 단어에 미세한 밑줄 표시 | 없음 (Tier 1 기본) |
| 역추적 칩 | 오답 1회 시 Trace +0 (1런당 1회 한정) | 없음 (1회용) |
| 과부하된 신경망 | 정답 시 Trace -20 (대폭 회복) | 오답 시 penalty_rate ×2 |
| 저잡음 인터페이스 | Mutator M-03(노이즈 인젝션) 완전 무효화 | NIGHTMARE 노드 penalty_rate +15 |
| 깨진 모래시계 | 타이머 +60초 | 오답 시 추가로 -15초 차감 |
| 아르고스의 파편 | 런 시작 시 target_keyword 첫 글자를 미리 공개 | 런 종료 시 획득 SP -30% |
| 메아리 프로토콜 | 이전 런에서 맞혔던 노드 재등장 시 target_keyword 자동 입력 | 해당 노드에서 SP 획득 없음 |

> **승수 근거:** 유물 조합이 런마다 달라지면 같은 노드가 다른 전략 문제가 된다. 최소 15종 유물로 경우의 수는 수천 가지.

---

### 2-4. 분기 맵 시스템—ROUTE MAP (×1.2 승수)

런을 선형이 아닌 **노드 선택 지도**로 구성한다.

```
[ START ]
    │
    ├─ Easy ─ Easy ─ Hard ──────────┐
    │                               ├─ NIGHTMARE BOSS ─ [ END ]
    ├─ Hard ─ Hard ─ NIGHTMARE ─────┘
    │
    └─ Easy ─ SHOP ─ Hard ─ REST NODE ─ NIGHTMARE BOSS
```

**특수 노드 타입 (신규 도입):**

| 타입 | 효과 |
|------|------|
| **SHOP** | 획득 SP로 유물 구매 / Trace Level 감소 |
| **REST** | Trace -30 회복. 단 다음 노드 penalty_rate +10 |
| **ELITE** | Hard 난이도 + 특수 유물 드롭. 기피 가능 |
| **MYSTERY** | 랜덤 효과 (유물 획득 or Trace +20 패널티) |

---

### 2-5. 데일리 챌린지 (무한 콘텐츠)

매일 서버 시드로 고정된 런 구성 제공. 전 세계 플레이어와 **같은 노드·같은 유물로** 경쟁.

- 전 세계 리더보드 (클리어 타임 / Trace 잔여량 / 사용 힌트 수)
- 일일 참여 보상 (SP, 전용 유물 언락)
- **클리어 조건:** "오답 없이 클리어", "GHOST 클래스로 5분 내 완주" 등 특수 조건

> 데일리 챌린지 1일 20-40분 × 365일 = **단독으로 120시간** 잠재.

---

### 2-6. 업적 / 챌린지 트리 (방향성 제공)

플레이어가 "다음에 무엇을 해야 하는가"를 항상 알 수 있게 한다.

**업적 예시:**

| 카테고리 | 업적명 | 조건 | 보상 |
|---------|--------|------|------|
| 탐험 | 첫 번째 오염 | 첫 번째 사망 | 칭호: 유기체 |
| 탐험 | 완전 침묵 | 오답 없이 런 클리어 | 유물: 순수 신호 |
| 도전 | 아르고스에게 경의를 | NIGHTMARE 보스 5개 모두 클리어 | 캐릭터 스킨: 아르고스 터미널 |
| 극한 | 검은 화면 속에서 | ASCENSION 20 클리어 | 비밀 엔딩 해금 |
| 컬렉션 | 기억 수집가 | 유물 50종 획득 | 아트 갤러리 해금 |

**총 목표 업적 수: 100+개** → 각 업적이 방향성을 제공하며 런 동기가 됨.

---

### 2-7. 멀티 엔딩 구조 (스토리 완주 동기)

단순 "클리어" 외에 **조건부 엔딩**으로 여러 번의 완주를 유도.

| 엔딩 | 해금 조건 |
|------|---------|
| 엔딩 A: 신호 소멸 | ASCENSION 0 일반 클리어 |
| 엔딩 B: 메아리 | 오답 0회 런 완주 |
| 엔딩 C: 아르고스의 눈 | ASCENSION 10 이상 클리어 |
| 엔딩 D: 의미론적 붕괴 | 모든 NIGHTMARE 보스 오답 없이 클리어 |
| **엔딩 E: 터미널의 침묵** | **ASCENSION 20 + 전 클래스 클리어** (진엔딩) |

## LAYER 3 — 100시간 타임라인 검증

```
런 1회 평균 소요: 45분
────────────────────────────────────────────
[스토리 첫 완주]          2~3런     × 45분 = ~2시간
[클래스 3종 각 클리어]    9런       × 45분 = ~7시간
[유물 수집 & 빌드 탐구]   20런      × 45분 = ~15시간
[ASCENSION 1~10 도전]    30런      × 50분 = ~25시간
[ASCENSION 11~20 도전]   40런      × 60분 = ~40시간
[데일리 챌린지 참여]      60일분    × 30분 = ~30시간
[업적 & 멀티 엔딩 수집]   ---                ~15시간
────────────────────────────────────────────
합계:                                   ≈ 134시간 ✓
```

## 개발 우선순위 로드맵

```
Phase 1 (MVP)       ─ 노드 18개 / 기본 루프 / 아르고스 대사
Phase 2 (v0.5)      ─ 노드 40개 / 맵 시스템 / 유물 10종
Phase 3 (v1.0)      ─ 노드 80개 / 클래스 3종 / ASCENSION 10단계
Phase 4 (v1.5)      ─ 노드 120개 / ASCENSION 20단계 / 데일리 챌린지
Phase 5 (v2.0)      ─ 멀티 엔딩 / 업적 100종 / 리더보드
```

---

## 핵심 결론

> **노드 120개 × 시스템 승수 2.0 = 100시간**
>
> 노드를 1,000개 만드는 것보다,
> 80개 노드를 **다른 클래스 / 다른 유물 / 다른 Ascension 레벨**로
> 반복 플레이하게 만드는 것이 100시간을 달성하는 현실적 경로다.
> 데일리 챌린지는 그 자체로 서비스 기간 내 무한 콘텐츠 공급원이 된다.
